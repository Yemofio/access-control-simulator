# Access Control Simulator

Enterprise-grade Zero Trust, RBAC, ABAC, and MAC simulation with real-time threat detection.

## Features
- **Zero Trust Policy Engine**  
- **JWT Authentication** (RS256)  
- **Risk-Based Adaptive Access**  
- **Audit Logging**  
- **React + Flask Stack**
